# Daydreams Sandbox

A learning journey through the [Daydreams](https://daydreams.systems) / Lucid Agents ecosystem, with special focus on the x402 payment protocol.

## Learning Goals

- Build AI agents with Lucid Agents framework
- Monetize agents using x402 protocol
- Understand agent-to-agent (A2A) communication
- Explore on-chain identity with ERC-8004

## Projects

| # | Project | Concepts | Status |
|---|---------|----------|--------|
| 01 | [Wisdom Agent](./projects/01-wisdom-agent) | Lucid basics, x402 payments | In Progress |
| 02 | Multi-Tool Agent | Multiple entrypoints, pricing tiers | Planned |
| 03 | Agent Discovery | A2A protocol | Planned |
| 04 | Verified Agent | ERC-8004 identity | Planned |
